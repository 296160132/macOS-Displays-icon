# macOS-Displays-icon
[English Version](./en.md)  
## 关于此项目 
本项目是一个显示器设备图标数据库. 用于在macOS系统的"关于本机" > "显示器"界面中显示真实的设备显示器外观图标.

## 图标样式规则
1. 图标尺寸为 1024*1024 
2. 请使用您能找到到的最清晰的显示器外观图片
3. 显示器外观图像背景干净, 轮廓清晰, 无水印
4. 使用项目中的 "Screen.jpg" 作为显示器的显示内容
5. 需要为显示器基座添加适当的阴影
6. 位置坐标得体适中(正中偏下)
